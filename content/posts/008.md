### 具体情况

- 1、查询"01"课程比"02"课程成绩高的学生的信息及课程分数

```sql
select
    stu.*,
    sca.s_score as 01_score,
    scb.s_score as 02_score
from
    Student  stu ,
    Score sca ,
    Score scb
where
    stu.s_id = sca.s_id
    and stu.s_id = scb.s_id
    and sca.c_id = '01'
    and scb.c_id = '02'
    and sca.s_score > scb.s_score
```

- 3、查询平均成绩大于等于60分的同学的学生编号和学生姓名和平均成绩

```sql
select
    stu.s_id,
    stu.s_name,
    round(avg(sc.s_score),2) as avg_score
from
    Student stu
join
    Score sc
on
    stu.s_id = sc.s_id
group by
    stu.s_id,
    stu.s_name
having
    avg_score >= 60
```

- 4、查询平均成绩小于60分的同学的学生编号和学生姓名和平均成绩

- 53、交换相邻两行

```sql
-- 使用 (id+1)^1-1 计算交换后每个学生的座位 id.
-- 然后连接原来的座位表和更新 id 后的座位表.
-- COALESCE()依次参考输入的表达式,直至为非null则返回
SELECT
    s1.id, COALESCE(s2.student, s1.student) AS student
FROM
    seat s1
LEFT JOIN
    seat s2
ON
    ((s1.id + 1) ^ 1) - 1 = s2.id
ORDER BY
    s1.id;
------------------------------------
SELECT
    (CASE
        WHEN MOD(id, 2) != 0 AND counts != id THEN id + 1
        WHEN MOD(id, 2) != 0 AND counts = id THEN id
        ELSE id - 1
    END) AS id,
    student
FROM
    seat,
    (SELECT
        COUNT(*) AS counts
    FROM
        seat) AS seat_counts
ORDER BY
    id ASC;
```
