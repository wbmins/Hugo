# 单例模式(Sigleton)

> 保证一个类仅有一个实例,并提供一个访问它的全局访问点.

## 1、饿汉式(在类初始化直接创建不管你需不需要)

1. 直接实例化(间接直观)

```java
/**
 * ①饿汉式
 * 是否 Lazy 初始化：否
 * 是否多线程安全：是
 * 实现难度：易
 */
public class Sigleton1 {
    private static final Sigleton1 INSTANCE = new Sigleton1();
    private Sigleton1(){
    }
    public static Sigleton1 getInstance(){
        return INSTANCE;
    }
}
```

2. 枚举式(最简洁)

```java
/**
 * ②枚举:表示该类型的对象是有限的
 * 是否 Lazy 初始化：否
 * 是否多线程安全：是
 * 实现难度：易
 */
public enum  Sigleton2 {
    INSTANCE
}
```

3. 静态代码块(适合复杂实例)

```java
/**
 * ④懒汉式 静态代码块 (适合复杂实例)
 * 是否 Lazy 初始化：是
 * 是否多线程安全：是
 * 实现难度：一般
 */
public class Sigleton3 {

    private static final Sigleton3 INSTANCE;
    static {
        INSTANCE = new Sigleton3();
    }
    private Sigleton3(){
    }
    public static Sigleton3 getInstance(){
        return INSTANCE;
    }
}
```

## 2、懒汉式(延迟创建实力对象)

1. 线程不安全(单线程)

```java
/**
 * ③懒汉式
 * 是否 Lazy 初始化：是
 * 是否多线程安全：否
 * 实现难度：易
 */
public class Sigleton4 {
    private static Sigleton4 instance;
    private Sigleton4(){
    }
    public static Sigleton4 getInstance(){
        if(instance == null)
             instance = new Sigleton4();
        return instance;
    }
}
```

2. 线程安全(多线程)

```java
/**
 * ⑤懒汉式
 * 是否 Lazy 初始化：是
 * 是否多线程安全：是
 * 实现难度：易
 */
public class Sigleton4 {
    private static  Sigleton4 instance;
    private Sigleton4(){
    }
    public static Sigleton4 getInstance(){
        synchronized (Sigleton4.class){
            if(instance == null)
                instance = new Sigleton4();
            return instance;
        }
    }
}
//优化后
public class Sigleton4 {
    private static  Sigleton4 instance;
    private Sigleton4(){
    }
    //DCL double check lock 双端检锁机制
    public static Sigleton4 getInstance(){
        if(instance == null){
            synchronized (Sigleton4.class){
                if(instance == null)
                    instance = new Sigleton4();
            }
        }
        return instance;
    }
}
//更严格
public class Sigleton4 {
    //volatile禁止指令重排
    private static volatile Sigleton4 instance;
    private Sigleton4(){
    }
    //DCL double check lock 双端检锁机制
    public static Sigleton4 getInstance(){
        if(instance == null){
            synchronized (Sigleton4.class){
                if(instance == null)
                    instance = new Sigleton4();
            }
        }
        return instance;
    }
}
```

3. 静态内部类(多线程)

```java
/**
 * ⑥静态内部类
 * 是否 Lazy 初始化：是
 * 是否多线程安全：是
 * 实现难度：一般
 * 在内部类加载时才创建对象
 * 静态内部类不会随着外部类初始化
 * 而初始化的,他是要单独加载和初始化的
 */
public class Sigleton3 {

    private Sigleton3(){
    }
    private static class Inner{
        public static final Sigleton3 INSTANCE = new Sigleton3();
    }
    public static Sigleton3 getInstance(){
        return Inner.INSTANCE;
    }
}
```
