# IDEA

## 方法间加分割线

- File | Settings | Editor | General | Appearance | Show method separators

## 模板

### 类模板

- File | Settings | Editor | File and Code Templates | File Header 

```java
/**
 * @Classname ${NAME}
 * @Description TODO
 * @Date ${DATE} ${TIME}
 * @Created by ${USER}
 */
```
