### 启动方式

- IDEA启动

- 先打成jar包,在运行

    - mvn install 

    - java -jar xxx.jar 

- 根目录运行 mvn spring-boot:run (推荐)

    - 首次运行有点慢,需要加载插件(**可以走代理**)
